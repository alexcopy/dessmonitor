DEVIATIONS FROM PLAN
One minor deviation from the PLAN.md Section 4.4 script design:

- Added sys.path manipulation to insert the repository root directory
  into sys.path at the start of main(). Without this, 'app', 'shared_state',
  and 'service' are not on the Python path and importlib.import_module()
  fails with "No module named 'app'" for all modules. The CI environment
  may have the working directory on PYTHONPATH by default, but locally
  and for robustness, the explicit path insertion is necessary.

  Added lines:
    _REPO_ROOT = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
    if _REPO_ROOT not in sys.path:
        sys.path.insert(0, _REPO_ROOT)

  This is a purely additive fix for path resolution. No module names,
  classification, or validation logic was changed. The script is otherwise
  identical to PLAN.md Section 4.4.

All other aspects match the approved plan exactly.
